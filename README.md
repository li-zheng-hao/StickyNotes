## 一些特性 Features

1. 支持字体放大和缩小
2. 开机自启动
3. 窗口置顶
4. 定时保存
5. 窗体靠上边缘自动吸附隐藏，鼠标上移显示
6. 支持多个窗体主题
7. 全局快捷键功能
8. 自动更新
